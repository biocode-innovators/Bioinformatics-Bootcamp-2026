# 🧬 Lecture 05 – DNA Sequence Analysis

This lecture introduced the fundamental concepts of **DNA Sequence Analysis**, an essential component of Bioinformatics used to interpret nucleotide sequences and identify biologically significant features. Participants learned how sequence composition and computational analysis provide valuable insights into gene structure and function.

## 📚 Topics Covered

- Introduction to DNA Sequence Analysis
- Understanding DNA Sequence Composition
- GC Content and AT Content Analysis
- Biological Significance of GC Content
- Calculating Sequence Length
- Identification of Start and Stop Codons
- Open Reading Frames (ORFs)
- Sequence Translation using the ExPASy Translate Tool
- Applications of DNA Sequence Analysis in Genomics and Molecular Biology

By the end of this lecture, participants gained a foundational understanding of how to analyze DNA sequences, calculate nucleotide composition, identify coding regions, and translate nucleotide sequences into proteins using bioinformatics tools.
